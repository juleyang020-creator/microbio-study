## 二、运行与开发环境

- **开发语言**：HTML5 / CSS3 / JavaScript（ES5 语法，原生实现，无第三方框架）。
- **架构**：纯前端单页应用（SPA），数据内置于程序（`data/*.js`），无服务器、无数据库；核心内容可离线运行，外部参考链接仅在用户主动点击后访问；通过 Service Worker 实现离线缓存（PWA，可"添加到主屏幕"）。
- **运行环境**：任意支持现代浏览器的设备（Windows / macOS / iOS / Android）；亦封装为 iOS WKWebView 原生壳应用。
- **开发工具**：文本编辑器 + Node.js（仅用于运行单元测试，不参与运行）；iOS 端使用 Xcode。

## 三、软件功能结构

软件由"九大知识模块 + 四个独立工具 + 两个对比工具 + 权威参考"构成：

**知识模块**（左侧分类树浏览，点击查看详情）
1. 微生物分类（177 种细菌/真菌/病毒/寄生虫）
2. 抗微生物药
3. 耐药因素（耐药机制）
4. 鉴定卡（VITEK 等鉴定卡原理）
5. 药敏卡（药敏卡组成）
6. 试验（药敏试验、耐药表型检测、鉴别试验）
7. 染色（常规染色、特殊染色、真菌/寄生虫染色）
8. 培养基
9. 生化反应（原理与示意图）

**微生物详情页内容**（每个菌种按规范呈现）：形态与染色 → 致病性 → 天然耐药 → 形态（镜下+培养）→ 药敏折点（CLSI）→ 相似菌与鉴别 → 生化反应 → 治疗要点 → 关联条目 → 综述/参考（权威来源链接）。

**速查工具**
1. **天然耐药速查**：聚合所有微生物的天然耐药，按菌属分组，支持模糊过滤。
2. **异常药敏速查**：常见需修正/警示的药敏结果组合（按"必须修正/需复核/限制报告"分级）。
3. **关联关系图**：以任一条目为图心，SVG 绘制 1~2 层关联网络；可点击节点重设图心或进入详情。
4. **折点判读**：输入 MIC 数值自动判读 S/I/R；支持按菌组/药物筛选 CLSI 折点。
5. **生化对比 / 药敏卡对比**：多菌并排比较生化反应或药敏卡组成。

**搜索**：支持中文名 / 拉丁名 / 英文 / 词根的中英混合分词检索。

## 五、技术特点

1. 核心内容纯本地离线运行，不收集任何用户数据；外部参考链接仅由用户主动点击打开。
2. 药敏折点严格依据 CLSI M100 / M45 / M60 标准整理，并标注来源表号。
3. 内容多源交叉核对（CLSI、IDSA、CDC、Sanford、StatPearls、默沙东、LPSN/NCBI）。
4. 关系图采用同心圆 + 径向树自适应布局算法，自动减少连线交叉。
5. MIC 折点自动判读引擎，支持无敏感折点（仅中介/耐药）等特殊情形。
6. 全部内容渲染采用安全 DOM 构建（不使用 innerHTML），防注入。
